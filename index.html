About Me

Writing about yourself is a bit awkward, the hardest part is figuring out how to start talking about yourself - but I'll give it shot. Professionally, my career in coding has only beginning but I am incredibly excited about this field. The possibities of code seem endless and I've found it difficult to put the assignments down because it's such a fun puzzle. However, I'm leaving my options open moving forward from this class because there are so many avenues left to explore.
